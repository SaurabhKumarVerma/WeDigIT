## Clone project

```bash
- [Github Project Link](https://github.com/SaurabhKumarVerma/WeDigIT)
cd WeDigIT
```

## Step 1: Install dependencies

```bash
# using Yarn
yarn install

# or using
npm install

```

## Step 2: Start Application

### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

## IOS

```bash
cd ios
pod install
```

- [Expo](https://reactnative.dev/docs/environment-setup)
